## Step RUN

1. Clone repo dan masuk ke folder
    ```
    git clone https://github.com/yogik23/skw-3dos.git && cd skw-3dos
    ```
2. Install Module
    ```
    npm install
    ```
3. Submit `API key` dan `Bearer` di `data.txt`
    ```
    nano data.txt
    ```
   format data.txt
    ```
    apikey1:Bearer1
    apikey2:Bearer2
    apikey3:Bearer3
    ```
4. Jalankan bot 
    ```
    npm start
    ```

**Recomended to using vps to automatic claims**
